# Online Retail Data Analytics Case Study

## Business Problem
Analyze retail transaction data to identify best-selling products,
customer behavior, and peak shopping times to support business decisions.

## Dataset

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Key Analysis
- Bestselling products by quantity and revenue
- Sales trends by hour and day
- Repeat customer analysis
- Average Order Value (AOV)


## Key Insights
- A small number of products drive most revenue
- Sales peak during midday hours
- Repeat customers contribute disproportionately to revenue

## Business Recommendations
- Implement a loyalty program
- Schedule promotions before peak hours
- Create product bundles and cross-sell offers
